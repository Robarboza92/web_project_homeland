# web_project_homeland

Projeto de Patria a Patria, traz um pouco de informação sobre atores de diferentes localidades no mundo. O projeto possui introdução, um grid de fotos, cards individuais com informações de cada localidade com botoes ativos que levam a informações sobre NFT, e informação sobre o local de residencia do ator do projeto, alem de info sobre a galeria.
Neste projeto foi utilizado HTML para implementar as informações necessarias incluir botões. E com o CSS foi feita toda a estilização, detalhes, cores, inclusão de imagens de undos e posicionamento dos textos, imagens, icones, organização de elementos, alem de condicionantes, etc.
